## Semaine 2 — Presentation solution

### Post 4 — Lancement GRID
J'ai lance GRID : une extension Chrome qui pseudonymise automatiquement vos donnees quand vous utilisez l'IA.
100% local. Zero serveur. 30 secondes a installer.

### Post 5 — Demo technique
"M. Dupont" → "[Nom_1]"
"jean@company.com" → "[Email_1]"
"FR76 3000 6000 0112 3456 7890 189" → "[IBAN_1]"
Voila ce que GRID fait en temps reel quand vous collez dans ChatGPT.

### Post 6 — Pourquoi 100% local
Pas de serveur = pas de fuite possible.
GRID utilise Web Crypto API pour chiffrer le journal de conformite.
Meme moi, le developpeur, je n'ai pas acces a vos donnees.

## Semaine 3 — Cas d'usage

Le secret professionnel avocat-client ne s'arrete pas a la porte de ChatGPT.

Les donnees de sante sont Art.9 RGPD — protection renforcee.
