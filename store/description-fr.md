# DataMask – AI Data Privacy

## Vos données restent sur votre ordinateur. Point.

Des extensions Chrome ont espionné 8 millions de conversations IA en 2025.
DataMask fait exactement l'inverse : vos données ne quittent jamais votre navigateur.

## Comment ça marche

1. Vous copiez un texte avec des données sensibles (nom, email, IBAN, numéro de sécu...)
2. Vous le collez dans ChatGPT, Claude ou Gemini
3. DataMask pseudonymise automatiquement avant l'envoi — [Jean Dupont] devient [Personne_1]
4. Vous récupérez la réponse, DataMask vous montre la correspondance

## Ce qui est protégé

Emails, téléphones, noms et prénoms, IBAN et coordonnées bancaires,
numéros de sécurité sociale, clés API, adresses IP, données de santé (Art.9 RGPD)

## Sites protégés

ChatGPT, Claude, Gemini, DeepSeek, Copilot, Perplexity

## 100% local — ce que ça signifie concrètement

- Aucun serveur DataMask ne reçoit vos données
- Aucun compte à créer
- Le traitement se fait dans votre navigateur uniquement
- Journal de conformité RGPD chiffré localement (AES-256)
