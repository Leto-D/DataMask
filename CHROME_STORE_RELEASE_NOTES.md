   - Design moderne et responsive

### 📝 Description pour Chrome Web Store :

**Titre :** GRID - AI Prompt Builder

**Description courte :**
Extension Chrome d'aide à la création de prompts pour LLM - Outil de productivité pour créer des prompts efficaces

**Description détaillée :**
GRID est un outil de productivité conçu pour créer des prompts structurés et efficaces pour les modèles de langage IA (ChatGPT, Claude, etc.). 

Fonctionnalités :
✅ Templates prédéfinis pour l'analyse business (SWOT, KPIs, Lean Six Sigma)
✅ Interface intuitive en français
✅ Exemples concrets et professionnels
✅ Copie en un clic pour utilisation immédiate
✅ Aucune donnée collectée - 100% privé

Parfait pour les consultants, analystes business, formateurs et tous les professionnels utilisant l'IA dans leur travail.
